# nvidia-wayland-autofix

Recover a working GNOME + Wayland + NVIDIA setup after Ubuntu or kernel upgrades.

## Features

- Automatic NVIDIA driver detection (prefers open driver)
- Kernel headers + DKMS handling
- Safe dry-run mode
- Detect-only mode
- Detailed logging
- Wayland-first (no Xorg hacks)

## Usage

```bash
./nvidia-wayland-autofix.sh --help
```
